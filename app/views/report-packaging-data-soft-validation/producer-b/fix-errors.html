Packaging data file not uploaded – fix the errors and check the warnings

Your data file has not been uploaded to your organisation’s account. This is because some of the data in your file is missing or in the wrong format.

There are also 300 warnings. Warnings are unusual entries in your file you need to check, but they will not prevent your file from being submitted to the environmental regulator.

How to fix errors and check warnings

  1. Download your error report (CSV, 120KB). This tells you where the errors and warnings are in your file.
  2. Use the report to fix the errors. You may also need the file specification guidance (opens in a new window).
  3. Check the warnings and decide if you want to update any data.
  4. Upload the file again.

How the error report works

Your error report only shows the first 200 errors in the file you uploaded.

You should fix the errors, check warnings and upload the file again. If there are more errors, you may get another error report when you re-upload the file.
