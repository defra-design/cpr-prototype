Upload data for 2023 Upload data file to your organisation’s account

Before you can submit your 2023 data file to the regulator, you need to upload it to your organisation’s account.

When you upload the file we’ll check your data is in the right format - this is called ‘validation’.

If the data is OK, we’ll store the file on your organisation’s account.

You can update the file and upload it again as often as you need to before submitting, but we cannot store more than one version of the file on the account.

If there’s a problem with your file, we’ll reject the upload and tell you how to fix it.

Before you start

Read the guidance on how to upload packaging data as a compliance scheme (opens in a new window).

The guidance contains a blank version of the data file you need to use - fill it in with your organisation’s data.

Upload file

The file may take a few minutes to upload.

Do not close the window while the file is uploading.
