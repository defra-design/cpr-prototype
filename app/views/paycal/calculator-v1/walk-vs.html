Getting started with VS code with Databricks extension

First steps

Sign in to Databricks and make sure you're in the Data Science and Engineering workspace.


You can change your workspace on the left sidebar. Then select Users.

In the Users folder you will see your username at the top. This is your personal space - no one else will have access to it. A sandpit area - where you can test things.

When you are in this folder - create a new Notebook.


Create a Notebook

  1. Click the dropdown next to your name and select Create and Notebook.


  2. Enter a name and select a language. Select a cluster. If you can't see one - you can create one later. Click on Create.


  3. Your new Notebook will open - and you can start working.


  4. Click on the Run all button on the top bar to run your code.
