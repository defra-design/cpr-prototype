Using R, Python and SQL in Databricks

6. Visualize your data

Before you start - you will need your data loaded in to your Notebook.

If you haven't loaded it - this is explained in Load your data


Databricks offers a quick way to visualize your data for recognised data types.
This includes most tabular data types including data frames or similar.

Use the built in display() function to show your data in an interactive tabular form. This function is available in Python and R.

If you're using SQL - your data will automatically be displayed when you execute a query.

To see a graphic display of your data - click on the plus icon in the tab and select Visualization.

In the visualization editor - choose your visualization type and associated parameters and click on the Save button. That will save your graphic in to your notebook.


The generated plot is interactive. If you hover over a plot - additional information is displayed.


If you need to edit the plot - click on the Edit visualization button to edit the graph in the previous screen.
