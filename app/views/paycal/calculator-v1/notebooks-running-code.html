Select Workspace from the side menu and select Workspace and Users.

This is your personal folder - no one else will have access to it. Use this area to test things out.

Create a notebook in this folder.


Create a notebook in your workspace

  1. Click on the dropdown button on the right and select Notebook.


  2. Your new notebook will be called 'untiltled' and will have a default coding language. Click on the name to change it to a more meaningful one. Select the language you want to work in by clicking on the dropdown


  3. Click on the dropdown on the top right - to select a cluster. It might say Connect or Terminated.

    4. If you're using Python or SQL - you can use a shared cluster.
    If you're using R it's a bit more complicated. You will need to create a personal cluster.

    It's important to know how clusters work
  4. When you've connected to a cluster - type your code in the command cell.


  5. To use the helpful 'copy code' function below. Select the language tab and click on the copy code button. Paste this in to your command cell.

    6. Contents

    • Python
    • R
    • SQL 
    
                        print("Hello World")

  6. Click on the Run all button on the top bar to run your code.

    7. You may need to wait for your cluster to run.


  7. You can add code cells if you hover near the bottom of a cell and click on the plus icon. This will help to structure large amounts of code and make it easier to navigate.
