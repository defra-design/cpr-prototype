3. Load your data

Find the path to your data

Before you start - you will need to know the path to your data to be able to load it.
This will be a three part name with a catalog, schema and table.

Example: preprod_catalog_p02.ada_demo.csc_data

Select Data from the left side menu.


  1. The data you have access to will be displayed here.


  2. Find your data and select it.
  3. Click on the Create dropdown button and select Notebook


  4. Type in the load commands related to your language.


  5. Run your code.
