Create subsidiary IDs

Use the file upload to create subsidiary IDs – you will need these subsidiary IDs to add subsidiaries to your organisation.

Before you upload your file, make sure you’re using the latest version of the subsidiary file template (opens in a new tab).

Adding and removing subsidiaries

To add or remove subsidiaries, you must update and resubmit your registration file within 28 days of the change. You will need to enter the subsidiary IDs to do this.
