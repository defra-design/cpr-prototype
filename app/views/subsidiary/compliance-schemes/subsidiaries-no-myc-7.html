Subsidiaries

If you need to add a subsidiary to this parent company, use the file upload to create a subsidiary ID and then update and resubmit your registration file.

Before you start

Before you upload your file, make sure you’re using the latest version of the subsidiary file template (opens in a new tab).

Use the file upload to create subsidiary IDs – you will need these subsidiary IDs to add subsidiaries to your organisation.

Adding or removing members might affect your organisation's packaging obligations – make sure you have read the guidelines for extended producer responsibility for packaging (opens in a new tab).

Upload a file

The file must be:

  • smaller than 60KB
  • in CSV format

It may take a few minutes for your file to upload - do not close the window while the file is uploading.
